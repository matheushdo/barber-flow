# 💈 Barber Flow

Sistema de gerenciamento para barbearias, desenvolvido para facilitar o controle de clientes, serviços e agendamentos.

## 🚀 Tecnologias utilizadas

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express

### Banco de dados
- MySQL

### Ferramentas
- Git e GitHub
- Insomnia
- MySQL Workbench
